Introduction

C is a general-purpose, procedural, imperative computer programming language developed in 1972 by Dennis M. Ritchie at the Bell Telephone Laboratories to develop the UNIX operating system. C is the most widely used computer language . It keeps flactuating at number one scale of popularity along with Java programming language, which is also equally popular and most widely used among modern software programmers.The C programming language is a structure oriented programming language.
