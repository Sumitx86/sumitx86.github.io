Andrew Hoyer/Experiments

Every letter you see in this simulation is actually an HTML div element being moved around using CSS transforms and some simple physics.

Be sure to check out my blog post about this experiment to learn how I made it all happen.

This experiment should work in all modern browsers (although I have not tested IE9). Seems to perform best in Safari and Webkit Nightly builds.
